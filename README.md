# Web3 Transaction History Viewer Basic

This project helps beginners learn how blockchain transactions can be read and shown in a Web3 app.

You will learn:
- How to connect a wallet
- How to read recent transactions
- How to show transaction data
- How blockchain explorers work
- How Web3 apps fetch on-chain history

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-transaction-history-viewer-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- history.js = Transaction functions  
- web3.js = Blockchain connection  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Connect wallet  
- Show recent transactions  
- Display hash and value  
- Simple history list  

## Learning Goal
This project explains how wallet transaction history is read from the blockchain.
